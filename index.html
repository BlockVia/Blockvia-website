FAQ & info

Is Blockvia HOST free?

Yes. The launcher itself is free to use. You only need your own hardware or hosting machine to run the servers.

Do I need to install Java?

No. Blockvia HOST can download and manage a local Java runtime for you, so you can focus on the server itself.

Which Minecraft versions are supported?

The launcher reads the official Mojang version manifest for Java and a maintained list for Bedrock, so you always see up‑to‑date versions.

What platforms are supported?

The current build targets Windows 10/11 64‑bit. Other platforms may be added later.

About Blockvia HOST

Blockvia HOST was designed to feel like a modern Minecraft launcher while still giving you full control over your own servers. It uses a clean Electron interface with a console, world editor, player manager and file browser in one window.
